First launch on macOS

macOS will block the app the first time you open it ("Apple could not verify TranscribeTool is free of malware…"). That's because we haven't paid Apple's $99/yr signing fee yet — the app itself is fine. Two ways to bypass:

Option A · Settings
  1. Close the warning dialog.
  2. Open System Settings → Privacy & Security.
  3. Scroll to Security.
  4. Find "TranscribeTool was blocked…" → click Open Anyway.
  5. Re-open the app and click Open.
Option B · Terminal

One command, then open normally:

xattr -dr com.apple.quarantine /Applications/TranscribeTool.app

You only do this once — every subsequent launch is a normal double-click.

What it does

Three tools that work together. Or separately.

Parse a channel

Paste a handle or URL. Get every video URL — regular, shorts, and live streams — into one list. Filter by tab, limit per tab for quick testing.

Download audio

Video or audio-only (~10× smaller). Resumes interrupted runs, skips videos already downloaded. Built on yt-dlp.

Transcribe locally

Whisper runs on your machine — no cloud, no API key. 99+ languages. Apple Silicon gets mlx-whisper (2–3× faster); everything else uses faster-whisper.
